# 面向对象
## 抽象
世间万物皆对象，面向对象的设计也是基于现实世界的。

- 面向过程

体现了程序员的逻辑思维能力，其本质是把一个业务看作一个流程化操作动作（使用函数来实现），一步步的去执行。

- 面向对象

体现了程序员的抽象思维能力，其本质是把一个业务分成若干个模块（使用对象[类]来实现），由这些模块交互组合去执行。

- 生命周期

构造函数（实例化对象） -> 对象的使用/操作 -> 析构函数（销毁对象）

//面向过程编程，只有三步
